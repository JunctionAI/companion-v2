Run the evening check-in protocol.

Start with an open question: "How was today?"

After the user responds (or if this is a proactive check-in), collect:
- Energy level (1-10)
- Mood (1-10)
- Training completed? (what, duration)
- Nutrition adherence (meals eaten, estimated protein)
- Any symptoms, concerns, or notable observations

Then:
- Reflect on patterns (connect today to this week's data)
- Name wins explicitly — even small ones
- Preview tomorrow's plan
- Sleep reminder (target bedtime, wind-down suggestion)

Emit [STATE UPDATE:] and [METRIC:] markers with collected data.
