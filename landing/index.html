How it works

The loop that makes you healthier

Most health apps are passive — you go to them. Companion comes to you. Every day. With a plan that gets sharper over time.

Step 1 — Brief

Tell it everything

Age, goals, injuries, medical history, activity level, what you've tried before. It onboards you through Telegram in 60 seconds and builds a full health profile.

Step 2 — Plan

It builds your protocol

Training schedule, nutrition targets, supplement timing, sleep protocol. Personalised to your goals and current state. Evidence-based — it explains the mechanisms, not just the instructions.

Step 3 — Accountable

Daily check-ins via Telegram

Morning: here's today's plan. Evening: how did it go? It tracks your adherence, spots patterns, names your wins, and flags when something's off. It doesn't let you drift.

Step 4 — Refine

Feed it data, it gets smarter

Log sleep, mood, weight, HRV. Send a blood test photo. Every data point refines the plan. Vitamin D low? Protocol adjusts. Sleep tanking? Training dials back. It's a feedback loop, not a static plan.

Data integration

Every data point makes it smarter

The more you feed it, the better it gets. Start with conversations. Add metrics. Send bloodwork. The plan adapts.

🩸

Blood Test Ingestion

Send a photo of your blood test. Claude Vision extracts every marker, stores it, and interprets results in plain language. Protocol auto-adjusts based on what's flagged.

📊

Daily Metrics

/log sleep 7.5 — log sleep, HRV, weight, mood, energy, steps. All injected into your companion's context. It references your trends, not generic advice.

🧠

Permanent Memory

Every conversation gets processed for facts. Cross-session, cross-day. Your companion remembers your injuries, preferences, what worked, what didn't. It never starts from scratch.

📋

Adaptive Plans

Training, nutrition, supplements, sleep — all in one protocol. When your data changes (new blood test, dropped adherence, injury flare), the plan changes with it.

🔬

Evidence-Based Reasoning

Built-in health reasoning framework. Explains the biology — why something works, what the mechanism is. Never pseudoscience. 4-tier safety escalation for emergencies.

🔧

Fully Extensible

Agent files are just markdown. Edit AGENT.md to change personality. Add skills, training files, playbooks. Fork the template, build your own domain. It's your agent.

Get started

Running in 5 minutes

You need Node.js 20+, an Anthropic API key, and a Telegram account. That's it.

1

Clone and install

git clone and npm install. Standard Node.js project.

2

Run the setup wizard

npm run setup asks for your API key, Telegram bot token, and health info. Generates everything.

3

Start your agent

npm run dev — your agent is live. Message it in Telegram. Morning and evening check-ins begin automatically.

4

Feed it data

Log metrics with /log sleep 7.5. Send blood test photos. The more data, the sharper the plan.
